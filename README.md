**Supervisor:** Ms. Seema Dahiya | **Turnitin Score:** 4%

## Overview
This research investigates ML techniques for early disease prediction using the Pima Indians Diabetes and Cleveland Heart Disease datasets. Seven algorithms were tested and compared.

## Key Result
**XGBoost achieved the highest accuracy — 94.5% with AUC-ROC of 0.94**

## Algorithms Compared
Logistic Regression, Decision Tree, Random Forest, SVM, Naive Bayes, KNN, XGBoost

## Tools Used
Python 3.10 · Scikit-learn · XGBoost · SMOTE · Pandas · NumPy · Matplotlib · Jupyter Notebook
